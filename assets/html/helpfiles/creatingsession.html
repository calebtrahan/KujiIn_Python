Creating Sessions

There are 2 parts to sessions that you can create.

  1. Entrainment (See below)
  2. Ambience (Covered in the ambience section)

Steps To Creating A Session:

  1. Start by dragging the sliders on the left to increase/decrease the number of minutes for each cut
  2. If you want you can add a pre (PRE) and post (POST) session. These are just contemplative meditation before and/or after you work on the cuts.
  3. Since the brainwave entrainment slowly guides your mind towards a certain state (Attuned with the relative Chakras for each cut), there is a 2 minute ramp into and out of the session.
  4. After you have selected the cuts you have worked on, click the CREATE button at the bottom of the program. You will be presented with a summary of what you have selected, along with a couple of checks to make sure everything is good before starting to create the session

Guidelines For Session Creation:

  • There doesn't have to be a value for each cut. However, it is best to start with RIN (for 2-3 min and due a couple minutes of each cut before the cut you are doing the major work on.
  • Make sure the cuts you are working on connect. Meaning that if you want to work on Sha And Zai, you need to work on the cuts in between (Kai, Jin, Retsu) for at least a couple minutes in order for me to create the session due to the differences in frequencies between non-connecting cuts

If you are planning on using ambience, click the CLOSE button and then the Adding Ambience button and follow those instruction before clicking CREATE to create the session.
