Adding Ambience To The Project Directory:

In order for the program to use the ambience, it must be placed in a specific directory. Starting from where you installed the program (I will call PROGRAMDIR below), here is where to place the ambience.

NOTE: All Ambience Files Must Be In .mp3 Format To Be Recognized By The Program

There are 2 different options for ambience:

  1. General Ambience. This is a big (or small) list of files to be played for the whole session, regardless of which cut is playing.
  2. Specific Ambience. This is if you want to play ambience for each individual cut seperately. For example, if you have a certain file (or files) you want to listen to during RIN, but different files you want to listen to during KAI or any other cut, my program will play the ambience in that folder for each specific cut, giving a great deal of variety to the sessions you create.

General Ambience:

Place the files in: PROGRAMDIR\sound\ambience\general\

The more files you place in here the better. My program will play the files at random, so the more you have the more diverse the ambience in the session will be. If there are not enough ambience files to play unique files for the length of the session, the ambience will automatically loop in random order, so the more you have the more random and diverse the ambience will be

MUST HAVE AT LEAST 3 FILES IN THE PROGRAMDIR\sound\ambience\general\ IN ORDER TO USE GENERAL AMBIENCE

Specific Ambience:

Place The Files In:

PROGRAMDIR\sound\ambience\rin (These will be played while practicing RIN)

PROGRAMDIR\sound\ambience\kyo (These will be played while practicing KYO)

...and so on for each cut

Also, place files in the PROGRAMDIR\sound\ambience\presession and PROGRAMDIR\sound\ambience\postsession as well to use this feature

MUST HAVE AT LEAST 2 AMBIENCE FILES IN EACH DIRECTORY (INCLUDING presession AND postsession) IN ORDER TO USE THIS FEATURE

Adding Ambience To The Session:

With one of the ambience types (general or specific) guidelines met. Click the ambience checkbox in the bottom left. You will be presented with a dialog to select which type of ambience (specific or general) you would like and after clicking, the program will verify that the guidelines (ABOVE) are met. And if so, the ambience will be automatically added to the program when you click the CREATE button.

If The Ambience Box Stays Ticked, All Checks Were Passed And You Are Good To Go, The Ambience Is Successfully Added To The Session. Click CREATE and the session will create both the entrainment, and also randomized ambience that will play with the session when you click PLAY
